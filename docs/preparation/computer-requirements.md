# Computer Requirements and Software Tools

This page outlines the software and accounts you need for the course.

---

## Requirements Checklist

- [ ] Windows 11 laptop (or remote desktop access with admin privileges)
- [ ] Google account for cloud storage
- [ ] Draw.io account
- [ ] EDA Playground account
- [ ] AWS Academy account

---

## Computer Setup

### Laptop Requirements

It is desirable that you have a **Windows 11 laptop** when you come to the classroom. If not, you should have a way to interface with a remote desktop Windows computer where you have administrative privileges.

### Cloud Storage

Temple user accounts do not have Google Drive access. It is a good idea to request a **personal Google account** that you can use to store your course resources.

---

## Software & Accounts

### 1. Draw.io (Block Diagrams)

- **Web version**: [app.diagrams.net](https://app.diagrams.net/)
- **Desktop version**: [Download Draw.io](https://www.drawio.com/)

**Template Files**:
- Download [Scratchpad.xml](../etc/Scratchpad.xml)
- Download [ECE.xml](../etc/ECE.xml)

See the [Draw.io Guide](Drawio) for detailed usage instructions.

### 2. Class Notebook

Get an account from the class Notebook for group collaboration.

### 3. EDA Playground

Online Verilog simulator for quick testing:
- Sign up at [edaplayground.com](https://edaplayground.com/)

### 4. AWS Academy

Cloud development environment:
- Create account at [AWS Academy](https://awsacademy.instructure.com/)
- See the [AWS Setup Guide](aws-setup) for detailed instructions

---

## Next Steps

After completing the requirements above:
1. Follow the [AWS Setup Guide](aws-setup) to configure your cloud environment
2. Review the [Draw.io Guide](Drawio) for creating block diagrams
3. Download the template files for your assignments
