# Computer Requirements and Software Tools

1. It is desirable that you have a windows-11 laptop when you come to the classroom. If not, you should have a way to interface with a remote desktop windows computer that you have administrative privilege.

2. Temple user account does not have a google drive. It is a good idea to request a google email account that you can store your course resource at.

3. Get an account at [draw.io](https://app.diagrams.net/) or [download draw.io](https://www.drawio.com/)

4. Download [Scratchpad.xml](../etc/Scratchpad.xml) and [ECE.xml](../etc/ECE.xml) so you can use for draw block diagram

5. Get an account from class Notebook for group collaboration

6. Get an account from [EDA Playground](https://edaplayground.com/)

7. Setup an [AWS academy account](https://awsacademy.instructure.com/) - See the [AWS Setup Guide](aws-setup) for detailed instructions
